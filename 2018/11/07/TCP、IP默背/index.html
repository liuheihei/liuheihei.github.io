第三章 IP

特性

不可靠、无连接

路由选择

每个IP数据包的发送路径可能都不相同。

路由分为三类： 主机类路由、网段类路由、默认路由。

其优先级顺序为： 主机类路由>网段类路由>默认路由。

子网划分

子网掩码分为： 网络位、子网位、主机位。
划分子网的个数为 2的n次方。 n == 子网位
每个子网中主机的个数为 2的m次方减2 m==主机位

第四章 ARP协议

ARP: 将IP地址转换为Mac地址在链路层进行数据传送。

全1代表广播地址： mac地址是16进制。

ARP在本地主机会有一个ARP缓存，当一个需要访问某一个IP地址时，需要先在内部发起arp广播，确认对方的arp地址后，才进行数据发送。超时时间一般为75s.
