RHEL-Debian包管理文档

发表于 2017-01-16 | 分类于 技术 |

ubuntu 软件包使用

之前一直在RHEL系列工作，安装软件系列的操作十分熟悉了，比如rpm、yum命令都是使用的非常熟悉。但是debian的apt-get、dpkg确很生疏，今天有空记录一下RHEL下经常使用的操作，并对应Debian系列如何使用。

Debain

dpkg

安装软件

$ dpkg -i <package_name> #保留配置
$ dpkg -P <package_name> #不保留配置

卸载软件

$ dpkg -r <package_name>

查看包的相信信息

$ dpkg -s <package_name>

查询某文件是由哪个包生成的

$ dpkg -S <filename>

查询包里安装里哪些文件

$ dpkg -L <package_name>

查询某包是否被安装

$ dpkg -l <package_name>
$ dpkg -l | grep <package_name>

列出所有被安装的包

$ dpkg -l
apt-get命令

安装包

$ apt-get install <package_name>

指定安装包版本

$ apt-get install <package_name>=1.1.1

卸载软件

$ apt-get remove <package_name> #保留配置
$ apt-get remove --purge <package_name> #删除配置
$ apt-get autoremote #删除包以及其依赖的软件包
$ apt-get autoremote --purge #删除包以及其依赖的软件包并删除配置文件

删除本地缓存

$ apt-get clean
$ apt-get autoclean

更新和升级

$ apt-get upgrade <package_name>

查询某个文件哪个包存在

$ apt-file search <filename>

查询某个包在源中的版本

$ apt-cache show   <package_name>

RHEL下软件包管理

rpm 命令

安装软件

$ rpm -ivh <package_name>

卸载软件

$ rpm -e <package_name> ( 如果有依赖关系，yum命令会更简单一点。)

查询某文件是由哪个包生成的

$ rpm -qf <filename>

查询包里安装里哪些文件

$ rpm -qf <package_name>

查询某包是否被安装

$ rpm -q <package_name>

列出所有被安装的包

$ rpm -qa

升级一个包

$ rpm -Uvh <package_name>

$ rpm -Uvh --oldpackage <package_name> #降级
yum命令

安装包

$ yum install <package_name> #自动安装依赖包

安装包组

$ yum groupinstall <group_name>

卸载软件

$ yum remove <package_name>

删除本地缓存

$ yum clean all

查询源中有哪些包

$ yum list

更新和升级

$ yum update / yum upgrade  #全部更新

$ yum update <package_name> # 更新单个包以及所依赖的包

yum update 和 yum upgrade 的区别(google 找的并未测试)

yum upgrade == yum update --obsoletes
yum upgrade  会升级包到最新到版本，举个例子，foo 新版改名为 foo2，那么它会安装 foo2
并且把 foo 删除，而 yum update 只会升级 foo，不会将包更新到 foo2。
因此yum update 可以更保守的管理包，但是yum upgrade 会保持包的状态是最新的。

查询某个文件哪个包存在

$ yum provides "*/<filename>"
